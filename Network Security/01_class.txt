# Network Criteria (Security Service which are get by Security mechanism)

    - Deliverable ( Not include to the network security)
    - Accuracy
    - Secrecy (Cryptography, steganography Thats are security mechanism or method)
    - Integrity (Digital Signature)
    - Authenticity (Sender Authentication, can be used DS)
    - Non-repudation (DS)

# Digital Signature

    - DS Send with sender message To identify the sender Authentication.
    - DS is code which created with hash function.
